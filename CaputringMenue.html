Single Page Webanwendungen - Menue mit Capturing Phase

Erstellen Sie ein Funktion registerMenu, der Sie ein DOM Element übergeben können,
und dass auf diesem dann einen Klick Handler registriert. Wenn das entsprechende
DOM Element angeklickt wird, zeigt der registrierte Klick Handler ein Menü an der
Stelle an, an der der Klick durchgeführt wurde. Welches Menü angezeigt werden soll,
ist für die Aufgabe Ihnen überlassen und Sie können das Menü fest in die Methode
registerMenu einbauen.
Wird, nachdem das Menü angezeigt wird, ein Eintrag im Menü angeklickt, wird
auf der Konsole ausgegeben, welcher Menüeintrag ausgewählt wurde und das Menü
wird geschlossen. Wird auf eine beliebige Stelle außerhalb des Menüs geklickt, soll das
Menü geschlossen werden. Der Klick soll abgefangen werden, so dass durch diesen
nicht mehrere Effekte erzieht werden. Verlässt die Maus das Browserfenster, soll das
Menü ebenfalls geschlossen werden.
